### 3️⃣ Hypothesis Testing
Performed two statistical tests using Python and SciPy:

#### 🧩 Test 1 – Two‑Sample T‑Test (Lead Time vs Cancellation)
- **Purpose:** Check if longer lead times increase cancellation risk.  
- **Result:** Average lead time for canceled bookings is significantly higher.  
- **Conclusion:** **Reject H₀** → Lead time significantly impacts cancellations.

#### 🧩 Test 2 – Chi‑Square Test (Loyalty vs Cancellation)
- **Purpose:** Examine if guest loyalty affects cancellation behavior.  
- **Result:** Repeated guests show lower cancellation rates.  
- **Conclusion:** **Reject H₀** → Guest loyalty and cancellation behavior are dependent.

---

## 📊 Key Insights
- Longer lead times correlate with higher cancellation probability.  
- Loyal guests are less likely to cancel bookings.  
- These findings validate the patterns observed in Task 2 and Task 3 dashboards.
